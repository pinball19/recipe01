## 食材を詳しく分ける方法
食材をもう少し細かく分類して使うことで、同じカテゴリ内でも整理しやすくなります。

### 詳細な食材例

#### **肉類**
- 鶏肉、豚肉、牛肉、ひき肉、ベーコン、ハム

**例**：
- `主菜_鶏肉_照り焼きチキン.md`  
- `主菜_豚肉_ポークソテー.md`  
- `主菜_牛肉_ビーフステーキ.md`  
- `主菜_ひき肉_ミートボール.md`

---

- `主菜_エビ_エビフライ.md`
